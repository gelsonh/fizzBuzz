The Code

1# Import the FizzBuzz logic into the Flask route
2from flask import Flask, render_template, request
3from services.fizzbuzz_service import generate_fizzbuzz
4
5app = Flask(__name__)
6
7@app.route("/the-app", methods=["GET", "POST"])
8def the_app():
9results = []
10
11if request.method == "POST":
12fizz_value = int(request.form["fizz_value"])
13buzz_value = int(request.form["buzz_value"])
14stop_value = int(request.form["stop_value"])

16results = generate_fizzbuzz(fizz_value, buzz_value, stop_value)

18return render_template("the-app.html", results=results)
19
20
21# Build an array of values following the FizzBuzz rules
22def generate_fizzbuzz(fizz_value, buzz_value, stop_value):
23results = []
24
25for number in range(1, stop_value + 1):
26if number % fizz_value == 0 and number % buzz_value == 0:
27results.append("FizzBuzz")
28elif number % fizz_value == 0:
29results.append("Fizz")
30elif number % buzz_value == 0:
31results.append("Buzz")
32else:
33results.append(number)
34
35return results

Description

The app receives three values: fizz_value, buzz_value, and stop_value. These values come from the form on the app view through a POST request.

The Flask route reads each value from request.form, converts it into an integer, and passes the numbers into generate_fizzbuzz.

The FizzBuzz function loops from 1 through the selected stop value. If a number is divisible by both values, it adds "FizzBuzz". If it is divisible only by the first value, it adds "Fizz". If it is divisible only by the second value, it adds "Buzz". Otherwise, it keeps the number.

The resulting list is returned to the template as results, where each value can be displayed with a different visual style.
